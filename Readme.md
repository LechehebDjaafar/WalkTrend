## 🛠️ Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/username/WalkTrand.git
   ```
2. Navigate to the project directory:
   ```bash
   cd WalkTrand
   ```
3. Open the `index.html` file in your browser:
   ```bash
   open index.html
   ```
